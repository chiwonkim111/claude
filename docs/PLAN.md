## 1. 서비스 개요

### 1.1 배경 및 문제 정의

상시 채용 증가와 AI 리터러시 향상으로 직장인들은 이직 외에도 자신의 시장가치를 확인하려는 니즈가 높아졌다. 그러나 기존 서비스는 "이직 시점의 단발성 이력서 작성"에 머물러 있어 비구직 기간의 재방문을 유도하지 못하고 있다.

**핵심 문제 3가지:**

| 문제 | 설명 |
|------|------|
| **P1. 인지 장벽** | 과거 성과·수치 복기 어려움, 구체적 맥락 망각, 작성 에너지 소모 |
| **P2. 보상 장벽** | 이력 업데이트 즉시 보상 없음, 비구직 기간 접속 동기 부족 |
| **P3. 신뢰 장벽** | AI 가짜 이력서 범람, 채용 담당자의 검증 비용 증가 |

### 1.2 서비스 한 줄 정의

> 바쁜 직장인이 일하면서 커리어를 자연스럽게 기록하고, 이를 채용 경쟁력으로 직접 연결하는 **리멤버의 지속 사용 플랫폼**.

### 1.3 핵심 가설

이직을 결심한 순간에만 이력서를 쓰는 현재 패턴을 깨고, **평소 업무 흐름 속에서 성과를 지속적으로 기록**하게 하면 프로필 품질이 높아지고, 매칭 정확도와 채용 성공률이 함께 올라간다.

### 1.4 범위 (Scope)

| 구분 | 내용 |
|------|------|
| **In Scope** | 분기별 성과 체크, 실시간 마켓 밸류, 동료 상호 인증(베타), AI 채팅 로그 동기화, 커리어 패스 제안 |
| **Out of Scope** | 리멤버 기존 채용공고 탭 개편, 기업 채용 담당자 UI, 급여 외 복지 비교 기능 |

### 1.5 현재 vs 목표 상태

| 구분 | 현재 상태 | 목표 상태 |
|------|----------|----------|
| 프로필 완성도 | 이직 시점에만 업데이트 | 분기별 지속 업데이트로 상시 고품질 유지 |
| 사용자 재방문 | 구직 기간에만 집중 | 비구직 기간에도 분기 1회 이상 접속 |
| 이력 신뢰도 | AI 가짜 이력서 범람 | 동료 인증 기반 검증된 인재 DB 구축 |
| 수익 모델 | 기업 채용 광고 단순 노출 | 신뢰도 기반 프리미엄 매칭 + 커리어 패스 구독 |

## 2. 목표 및 성공 지표 (KPI)

### 북극성 지표 (North Star Metric)

> **"분기별 성과를 입력한 활성 사용자 수"**
> 이 지표가 높아지면 프로필 품질 → 매칭 정확도 → 기업 만족도 → 채용 성공률이 동시에 개선되는 선순환이 만들어진다.

### KPI 체계

| 구분 | 지표명 | 측정 방법 | 목표치 | 우선순위 |
|------|--------|----------|--------|---------|
| 습관화 (P1 해결) | 분기 성과 입력 응답률 | 알림 발송 대비 입력 완료 비율 | **40%** | 🔴 Critical |
| 습관화 (P1 해결) | 프로필 업데이트 비중 | 최근 3개월 내 업데이트 비율 | **+20%p** | 🔴 Critical |
| 보상 (P2 해결) | 마켓 밸류 리포트 조회수 | 업데이트 후 리포트 확인 사용자 수 | 업데이트 대비 **80%** | 🟡 High |
| 보상 (P2 해결) | 3개월 리텐션 | 기업 재방문율 및 사용자 재방문율 | **+15%** | 🟡 High |
| 신뢰 (P3 해결) | 동료 인증 마크 수락률 | 요청 대비 인증 완료 비율 | **50%** | 🟢 Medium |
| 신뢰 (P3 해결) | 인증 마크 보유 사용자 면접 전환율 | 인증 마크 유무 그룹 A/B 비교 | 마크 보유군 **+20%** | 🟢 Medium |
| 수익화 | 커리어 패스 유료 전환율 | Phase 3 전환 시 측정 | **5%** | ⚪ Phase 3 |

## 3. 사용자 페르소나

### 페르소나 1: 김민준 — 바쁘지만 커리어는 챙기고 싶은 재직자 (메인 페르소나)

| 항목 | 내용 |
|------|------|
| 이름 | 김민준 (가명) |
| 나이 | 32세 |
| 직업 | 마케팅팀 시니어 매니저 (재직 4년차) |
| 거주지 | 서울 |
| 기술 수준 | 보통 (앱 사용에는 익숙하나 복잡한 기능은 기피) |
| 주요 기기 | 스마트폰 (출퇴근 중 주로 사용) |

**목표 (Goals)**
- 이직을 당장 원하진 않지만 내 시장가치가 어느 정도인지 궁금하다
- 성과를 제때 기록해두면 나중에 이직할 때 편할 것 같다

**불편함 / Pain Points**
- 😤 분기 말에 "이번 분기에 뭐 했지?" 가 기억이 잘 안 난다
- 😤 이력서를 업데이트하려 해도 어디서부터 시작해야 할지 막막하다
- 😤 업데이트해도 당장 뭔가 달라지는 게 없어서 동기부여가 안 된다

**행동 패턴**
- 주로 사용하는 시간대: 출퇴근 시간(7~9시, 18~20시)
- 정보 탐색 방식: 알림 → 앱 진입 → 빠르게 완료 후 이탈
- 결정 기준: 간편함 > 즉각적 보상 > 브랜드 신뢰

> *"이직할 생각은 없는데, 내가 지금 얼마짜리인지는 궁금하더라고요. 근데 이력서 쓰려면 너무 귀찮잖아요."*

| 항목 | 내용 |
|------|------|

| 나이 | 28세 |

| 기술 수준 | 높음 (AI 도구 적극 활용) |

**목표 (Goals)**
- 3~6개월 안에 더 나은 조건의 회사로 이직하고 싶다
- GPT로 작성한 이력서가 너무 흔해서 차별화 포인트가 필요하다

**불편함 / Pain Points**
- 😤 2년간의 성과를 이력서에 담으려니 기억이 흐릿하다
- 😤 연봉 협상할 때 내 시장가치를 데이터로 제시하고 싶다
- 😤 AI 이력서가 너무 많아 채용 담당자가 신뢰하지 않는다

**서비스와의 관계**
- **이 서비스를 쓰는 이유**: 과거 AI 대화 기록 동기화로 이력서 초안 자동 생성, 동료 인증으로 신뢰도 확보
- **핵심 가치 제안**: 검증된 이력 + 데이터 기반 연봉 협상 자신감

> *"GPT로 이력서 쓰면 채용팀이 다 알더라고요. 진짜 내 성과를 증명할 방법이 필요해요."*

## 4. 사용자 여정 (User Journey)

### 시나리오 A — 비구직 재직자 (메인 플로우)

```
[분기 말 알림] → [앱 진입] → [AI 맞춤 질문 응답] → [성과 자동 구조화]
       │                                                       │
       ▼                                                       ▼
[마켓 밸류 즉시 확인] → [만족감/동기 획득] → [동료 인증 요청]
                                                               │
                                                               ▼
                                               [다음 분기 반복 → 프로필 신선도 유지]
```

| 단계 | 트리거 | 행동 | 시스템 반응 |
|------|--------|------|-----------|
| 분기 말 | 푸시 알림 수신 | 앱 열고 AI 질문에 답변 | 성과 자동 구조화 + 타임라인 업데이트 |
| 즉시 | 완성도 스코어 상승 | 마켓 밸류 리포트 확인 | 연봉 범위 상승 시각화 |
| 1주 후 | 동료 인증 요청 수신 | 동료 성과 인증 | 양방향 신뢰 마크 부여 |
| 다음 분기 | 알림 재수신 | 자연스럽게 반복 | 프로필 신선도 유지, 채용 제안 수신 증가 |

### 시나리오 B — 이직 준비 구직자

| 시점 | 행동 | 시스템 반응 |
|------|------|-----------|
| 이직 결심 | 이직 모드 ON + 희망 직무 설정 | 커리어 패스 갭 분석 리포트 즉시 생성 |
| 준비 중 | 과거 AI 채팅 로그 업로드 | 성과 자동 추출 → 경력기술서 초안 생성 |
| 완성 후 | 동료 인증 요청 + 인증 마크 획득 | 채용 담당자 검색에 우선 노출 |
| 채용 제안 수신 | 마켓 밸류 확인 후 협상 진행 | 데이터 기반 연봉 협상 자신감 제공 |

### 5.2 기능 상세 명세

#### FR-001. 분기별 성과 체크 (Quarterly Check-in)

- **설명**: 매 분기 말(3·6·9·12월) 푸시 알림으로 성과 입력 유도. 5분 이내 완료 목표.
- **트리거**: 분기 마지막 2주 시점 자동 발송, 또는 사용자 수동 진입
- **입력값**: AI 맞춤 질문에 대한 텍스트 또는 음성 응답
- **출력/결과**: 구조화된 성과 카드 생성, 타임라인 자동 추가, 완성도 스코어 상승
- **예외 처리**: 입력 중 이탈 시 임시 저장 → 재진입 시 이어서 작성 가능
- **완료 조건 (Acceptance Criteria)**:
  - [ ] 분기 말 2주 전 푸시 알림이 자동 발송된다
  - [ ] 직무·역할 기반 맞춤 질문 3~5개가 순차 제시된다 (스와이프 카드형)
  - [ ] 음성·텍스트 혼합 입력이 모두 지원된다
  - [ ] 입력 완료 시 프로필 완성도 스코어가 즉시 반영된다
  - [ ] 완료 화면에서 "내 연봉 범위 확인하기" CTA가 노출된다

---

#### FR-002. 실시간 마켓 밸류 (Real-time Market Value)

- **설명**: 프로필 업데이트 직후 동일 직무·경력·스킬 기반 시장 연봉 범위를 즉시 계산하여 시각화.
- **트리거**: 성과 체크 완료 또는 프로필 수동 업데이트
- **입력값**: 직무, 연차, 스킬, 학력, 인증 수, 최근 업데이트 날짜
- **출력/결과**: 연봉 범위(Min~Max), 채용 제안 수신 확률(%), 업데이트 전·후 변화량
- **예외 처리**: 동일 직무 샘플 수 부족 시 "데이터 집계 중" 안내 + 신뢰 구간 표시
- **완료 조건 (Acceptance Criteria)**:
  - [ ] 업데이트 완료 후 5초 이내 연봉 범위가 시각화된다
  - [ ] 채용 가능성 퍼센티지가 표시된다 (예: "채용 제안 받을 확률 73%")
  - [ ] 업데이트 전·후 마켓 밸류 변화량이 함께 표시된다
  - [ ] 동일 직무 타 사용자 대비 강점/약점 벤치마킹이 익명으로 제공된다
  - [ ] "내 연봉 10% 올리려면?" AI 분석 리포트 진입 버튼이 노출된다

---

#### FR-003. 동료 상호 인증 (Peer Verification)

- **설명**: 리멤버 명함 DB 확인된 전·현직 동료에게 성과 인증 요청. 인증 완료 시 해당 성과에 파란 체크 마크 부여.
- **트리거**: 사용자가 특정 성과 항목에서 "인증 요청" 탭
- **입력값**: 성과 항목 ID, 인증 요청 동료 선택 (명함 DB 기반)
- **출력/결과**: 동료 인증 마크 부여, 채용 담당자 뷰에서 신뢰도 점수 상승 및 우선 노출
- **예외 처리**: 같은 사람이 동일 사용자에게 과다 인증 시 가중치 제한, 신고 기능 제공
- **완료 조건 (Acceptance Criteria)**:
  - [ ] 명함 DB 확인된 실명 계정만 인증자로 지정 가능하다
  - [ ] 동료에게 "예/아니오" 형식의 인증 요청이 발송된다
  - [ ] 인증 완료 시 해당 성과에 파란 체크 인증 마크가 부여된다
  - [ ] 인증 취소 및 신고 기능이 제공된다
  - [ ] 동일인의 과다 인증 시 가중치 제한 로직이 적용된다

---

#### FR-004. AI 채팅 로그 동기화 (AI Log Sync) — Phase 2

- **설명**: GPT·Claude·Gemini 등 AI 대화 URL 또는 파일을 붙여 넣으면 프로젝트명, 역할, 성과 수치 등을 자동 추출하여 경력기술서 초안 생성.
- **트리거**: 사용자가 대화 URL 붙여넣기 또는 텍스트 파일 업로드
- **예외 처리**: 민감 정보 로컬 처리 옵션 제공, 서버 비저장 원칙 준수, 법무 검토 완료 후 출시
- **완료 조건 (Acceptance Criteria)**:
  - [ ] 대화 URL 붙여넣기 또는 텍스트 파일 업로드로 연동이 가능하다
  - [ ] NLP 기반 업무 관련 내용이 자동 필터링된다 (비업무 대화 제외)
  - [ ] 추출 항목 (프로젝트명, 역할, 기술, 성과 수치, 기간)이 사용자 확인 후 반영된다
  - [ ] 개인정보 필터링 안내 및 로컬 처리 옵션이 제공된다

---

#### FR-005. 커리어 패스 제안 (Career Path AI) — Phase 3 유료

- **설명**: 현재 프로필과 희망 직무/연봉 입력 시 갭 분석 리포트 생성 및 분기별 달성 가능한 커리어 미션 제시.
- **트리거**: 사용자가 희망 직무/연봉 설정
- **출력/결과**: 갭 분석 리포트, 분기 미션 카드, 미션 완료 시 마켓 밸류 상승 예측치
- **완료 조건 (Acceptance Criteria)**:
  - [ ] 희망 직무/연봉 설정 후 현재 갭 분석 리포트가 생성된다
  - [ ] 동일 직무 이직자들의 공통 스킬 벤치마킹 데이터가 제공된다
  - [ ] 분기별 달성 가능한 미션 카드가 제시된다
  - [ ] 미션 완료 시 마켓 밸류 상승 예측치가 미리 표시된다
